# 🏥 ArogyaNidhi

ArogyaNidhi is an Android healthcare application developed using Kotlin and Firebase.  
The app helps users find suitable government healthcare schemes and nearby hospitals based on their details such as income, family size, job type, and card category (APL/BPL).

---

✅ User Login & Signup

# 📱 Modules

## 🔐 Authentication Module
Handles user login, signup, and password reset using Firebase Authentication.

## 🤖 AI Recommendation Module
Recommends suitable healthcare schemes using rule-based AI logic.

## 🏥 Hospital Module
Displays nearby hospitals with district filtering and search functionality.

## 📍 GPS Location Module
Automatically detects user location and suggests nearby hospitals.

## 👨‍💼 Admin Module
Allows admin users to add hospitals to Firebase Realtime Database.



# 📂 Project Structure

~~~

- MainActivity.kt → Login Screen
- SignupActivity.kt → User Registration
- QuizActivity.kt → Collect User Details
- RecommendationActivity.kt → AI Recommendation
- ResultActivity.kt → Display Result
- HospitalActivity.kt → Hospital Search & GPS
- AddHospitalActivity.kt → Add Hospitals
- ProfileActivity.kt → User Profile
- SplashActivity.kt → Splash Screen

~~~

- find government healthcare schemes
- locate nearby hospitals
- access healthcare support quickly

using Android technology and Firebase services.
